Test de chi cuadrado¶

Presentaremos un anlisis de la independencia entre las variables 'Smoking' y 'Alcohol'. Para la misma se realizar un test de chi cuadrado con las siguientes hiptesis:

  • $H_0$ = 'Fumar y tomar alcohol son dos variables independientes'
  • $H_1$ = 'Existe dependencia entre fumar y tomar alcohol'

Se intentar probar que existe una dependencia, es decir, que $H_1$ es vlida. Para esto primero se proceder a analizar la tabla de frecuencia observadas y marginales para dichas variables.

Para poder realizar una prueba de independencia de chi cuadrado es necesario adems tener una tabla de las frecuencias esperadas. Mientras ms se acerquen los valores observados a los esperados, el valor del estadstico se acercara a 0 avalando $H_0$. Caso contrario, si los valores son muy dispares se rechazara la hiptesis nula.

# Observed frecuency
observed_frecuency = pd.crosstab(young.Alcohol, young.Smoking)

# Calculate the chi-square results through scipy special function 'chi2_contigency'
[value, p_value, g, frequency_array] = stats.chi2_contingency(observed_frecuency)

# Expected frecuency
expected_frecuency = pd.DataFrame(frequency_array, columns=observed_frecuency.columns, index=observed_frecuency.index)
# Add total per row and column
expected_frecuency['All'] = expected_frecuency.sum(axis=1)
expected_frecuency.loc['All']= expected_frecuency.sum()

Si comparamos el valor del estadstico chi-cuadrado experiemental con el valor crtico del modelo veremos que si tomamos un $ \alpha = 0.05 $ con 6 grados de libertad

$ \chi^2_{experimental}\ \gt \ \chi^2_{6, 0.05} $

$ 140.746\ \gt \ 12.592 $

Lo cual nos lleva la conclusin de que debemos rechazar la hiptesis nula por existir dependencia entre ambas variables. No se puede decir con exactitud que beber conlleve a fumar o viceversa, pero podemos inferir por lo analizado anteriormente que la dependencia entre las variables se debe a una influencia social.
